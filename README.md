# Assignment 2 – OOP (Python)

Dieses Programm simuliert ein virtuelles Haustier, mit dem man interagieren kann.
Der Nutzer kann das Haustier füttern, mit ihm spielen und es schlafen lassen.
Das Programm wird in Python ausgeführt und zeigt den aktuellen Status des Haustiers an.
Die Werte für Hunger, Glück und Energie bleiben dabei immer zwischen 0 und 100.
